# Real-Time Crime Insights: Anomaly Detection Using Deep Learning

## About the Project
This project detects suspicious and abnormal activities from video footage using deep learning and computer vision techniques.

## Features
- Real-time video anomaly detection
- Detection of suspicious activities
- CCTV video analysis
- Deep learning-based classification
- Computer vision processing

## Technologies Used
- Python
- YOLOv5
- OpenCV
- PyTorch
- Flask

## How to Run
1. Clone the repository.
2. Install the required dependencies.
3. Run the application.
